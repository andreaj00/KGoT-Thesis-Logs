- **Ault system**: 
  - `gpt-4o` (Full Tools configuration and Essential Tools configuration)
  - `gpt-4o-mini` (Essential Tools configuration)
  
- **Local system**: 
  - `gpt-4o` (ZeroShot)
  - `gpt-4o-mini` (ZeroShot and Full Tools configuration)
